# 💝 relay_07

## 4주차 개발

### **C. 댓글 기반 랭킹 시스템**

게시글에 달린 댓글을 분석하여 나와 가장 친밀한 유저 3명을 표시한다.

- 게시글에 댓글을 달거나 대댓글을 달 경우를 분석하여 친밀도를 높이거나 낮춥니다.

#### 변경사항

* 데이터베이스에 friendScore, comment 테이블 추가
* 친밀도 테이블 정보를 바탕으로 친한 유저 TOP3를 화면에 보이도록 함
* 댓글 작성 기능 구현
* 친밀도 업데이트 기능 구현
